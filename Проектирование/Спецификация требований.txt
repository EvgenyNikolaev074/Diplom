Спецификация требований

1.1 Общий взгляд на продукт

Интернет-магазин "Citadel" дает возможность пользователям без проблем приобретать интеллектуальные системы "Умный дом", в современном мире данная тема является актуальной и будет развиваться. С помощью данной системы можно подобрать люые "умные" системы для помещения, подобрать оптимальный комплекс интеллектуальных систем, отслеживать статус заказа и какие датчики установлены в помещении.

1.2.2 Классы и характеристики пользователей

Администратор - пользователь который управляет сайтом. Имеет свою учетную запись, которая дает возможность просматривать и редактировать данные в базе, редактировать сайт, выкладывать и удалять товары 
Потребитель - пользователь ползующийся системой в первую очередь для покупки интеллектуальных систем. Пользователь имеет возможность регестрироваться, без регистрации покупка на сайте не возможна. Так же пользовател может редактировать свои данные в личном кабинете, удалять учетную запись. На одну почту возможна регистрация только одного аккаунта.

1.2.3 Операционная среда

Торговая площадка "Citadel" будет представлена на сайте в сети интернет. Поддерживаемые браузеры: Google Chrome, Yandex, Safari. Должна присутсвовать Онлайн Касса для осуществления денежных переводов. Данные платежных систем не должны храниться в базе данных системы. Должно присутсвовать приложение для платформ Android версии 4.2.2 и выше, ios 9.0 и выше.

1.2.4 Структура сайта

На сайте должны присутсвовать следующиие страницы:
- Главная страница 
- Каталог
- О компании
- Контакты
- Форма регистрации
- Форма авторизации (для пользователей)
- Форма авторизации (для администратора)
- Форма обратной связи
- Редактор сайта
- Личный кабинет пользователя 
- Корзина
- Форма оплаты
- Страница для работы с базой данных

1.2.5 Ограничение дизайна и реализация 

Для создания сайта должны использоваться языки программирования: HTML5, CSS3, JavaScript, PHP, SQL. Макет сайта должен быть разработан в Adobe Photoshop или Figma. 

1.2.5.1 Главная страница сайта

Главная страница сайта должна представлять из себя приветсвие пользователя. В начале страницы должна быть "шапка" с логотипом и меню. Так же должны быть указаны контактный телефон и почта. Логотип должен находится в левой части экрана, по середине меню, с права контакты и кнопка обратной связи. 
Далее блок со слайдерами и небольшим описанием, чем занимается компания
Следующий блок должен включать в себя карточки товаров, которые были добавлены последние (новинки). Карточки товара должны состоять из фотографии товара, краткого описания, цены, кнопки подробнее и добавить в корзину
Следующий блок должен состоять из общего описания систем "Умный Дом" с фотографиями и описанием принципа работы.
Заключительный блок с контактами и дублирующимся меню сайта.

1.2.5.2 Каталог

Каталог представляет из себя список товаров. На данной странице так же должна присутсвовать "шапка" аналогично как на главной странице.
